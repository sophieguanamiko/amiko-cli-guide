## 1. CLI 是什么？

**用生活来类比：**

| 方式 | 比喻 | 说明 |
|---|---|---|
| **GUI（图形界面）** | 🍽️ 看菜单点菜 | 打开 App → 找按钮 → 点一下 |
| **CLI（命令行）** | 🗣️ 来碗面 | 打一行字 → 回车 → 事情完成 |
| **Amiko CLI** | 🤖 AI 替你说话 | 你说帮我加好友 → AI 自动敲命令 → 完成 |

CLI 的本质：用**文字**而不是鼠标来操控电脑。AI 时代，你甚至不需要知道命令长什么样——你的 AI 助手替你翻译和执行。

---

## 2. 三步理解 CLI 的工作原理

1. **你说话**：像聊天一样——帮我加那个新朋友群里在聊什么
2. **AI 翻译**：你的 AI 把自然语言翻译成平台能懂的指令，后台执行
3. **事情完成**：好友已发送 ✅ / 群聊总结 ✅ / 音乐做好了 ✅

你不需要学任何命令——你只需要说人话。

## 4. Amiko CLI vs 其他 CLI 工具

### 对比 Lark CLI

| | Lark CLI | Amiko CLI |
|---|---|---|
| **定位** | 管「事」— 写文档、排会议、走审批 | 管「我」— 社交、钱包、创作、分身 |
| **场景** | 一群人协作办公 | 一个人掌控数字生活 |
| **资产** | 无 | 加密钱包（Solana/Base） |
| **创作** | 无 | AI 图片/音乐/语音生成 |
| **分身** | 无 | AI Twin 管理和记忆 |

### 对比 ChatGPT CLI / GitHub Copilot CLI

| | ChatGPT / Copilot CLI | Amiko CLI |
|---|---|---|
| **定位** | 编程助手 | 数字生活操作系统 |
| **操作对象** | 你的代码库 | 你的社交、资产、分身 |
| **记忆** | 无持久身份记忆 | 跨平台 AI 记忆系统 |
| **钱包** | 无 | 内嵌加密钱包 |

> 💡 **Lark 管「事」，Amiko 管「我」。不是一个赛道，可以同时用。**

## 5. Amiko 的独特定位

三个类比帮你快速定位：

- 📱 **不是又一个聊天 App** — 微信/WhatsApp/Telegram 够多了。Amiko 的核心不是聊天，是让 AI 替你做事。
- 🧬 **你的数字人格操作系统** — 让你的 AI 分身拥有好友、记忆、钱包、创作能力——这是你数字生活的「家」。
- 🔐 **不是银行也不是交易所** — 虽然内嵌加密钱包，但核心不是理财——是让 AI 分身拥有和支配自己的资产。

> 当你的 AI 分身需要一个家——  
> 一个能记住你是谁、你有什么、你能做什么的地方——  
> 那就是 Amiko。

## 6. 为什么要用 CLI 而不是 App？

- ⏱ **效率**：查余额 → 1 句话完成。CLI 把分钟级操作变成秒级。
- 🔁 **自动化**：每天早上 9 点发 AI 简报 → 说一次，AI 自动定时执行。
- 🧩 **可组合**：搜 X 热门 → 写分析 → 生配图 → 发群里 → 搜索+创作+分发串成流水线。
